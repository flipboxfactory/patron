    Manage Tokens

    This service is used to manage one or many Token Records in Craft.

    • find( $identifier )
    • getQuery( $criteria )

    #find( $identifier )

    Returns an Token Record

    ArgumentAcceptsDescription
    $identifierstring, integer, Token RecordA unique Token Record identifier
    • Twig
    • PHP
    {% set token = craft.patron.manageTokens.find(1) %}

    Returns a Token Query.

    ArgumentAcceptsDescription
    $criteriaarrayAn array of Token Query criteria.
    • Twig
    • PHP

    id: 1
}) %}
